# Glossary

- **BYOK** — bring your own API key; user supplies provider credentials.
- **WAL** — SQLite write-ahead log journal mode.
- **Yellow / red risk** — UI tier for actions needing inline confirm vs modal / typed confirm; **must** be backend-enforced when trust gate is done.
- **Routed intent** — `router` output: tool name + JSON args, possibly from LLM classification.
- **IPC** — Tauri `invoke` bridge between webview and Rust.
- **NSIS** — Windows installer technology used by the Tauri bundle target.
- **Specta** — type export story for Rust ↔ TypeScript; see `docs/IPC_BINDINGS.md`.
