# Neph

Neph is a private, trust-first Windows desktop assistant built with Tauri, Rust, and React for one-user daily workflows.
Run it locally with `npm install` and `npm run tauri dev`.
Status is alpha: safety gates are strong but features are still in active development, so treat outputs and automations as supervised.

## Quality commands

```bash
npm run typecheck
npm run verify
```
